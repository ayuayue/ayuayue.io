    case 'c':
        // 省略break则跟下个处理逻辑相同

    default:
}

    'a' => 'A',
    'b','c','d' => 'B',
    default => 'Not Found'
}

8. 函数参数命名传递 – 传递参数更随意.不在意默认值跟参数顺序

旧版本

1
2
3
4

function method($who,$when=now(),$do){
    echo "$who $do at $when";
}
method('lisa',null,'Eat breakfast')

新版本

1
2
3
4
5
6
7

function method($who,$when=now(),$do){
    echo "$who $do at $when";
}
method(
	who : 'lisa',
    do : 'Eat breakfast'
)

**对比: ** 使用键值对的方式对应参数,可以忽略有默认值的参数和参数的顺序.更加简洁.
